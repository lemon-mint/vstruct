Introducing Vstruct

Vstruct is a Code Generation Based High Speed Data Serialization Tool

Key Features

Zero Copy Deserialization

Vstruct is designed to support Zero Copy Deserialization.

Zero Copy Deserialization makes it easy to handle large files with low memory overhead.

Code Generation

Vstruct uses code generation to generate optimized type-safe code. Currently, Go, Dart, Rust and Python versions are available.

Getting Started

The easiest way to use Vstruct is to use the online interactive compiler.
